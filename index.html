Projects

Wikipedia Mock Search -> Code On GitHub

This application is using Wikipedia's api system to do a search through their records. This was my first crack at a React.js application. I was able to simplify my build process by learning Webpack; which handled ES6 transpiling, as well as the JSX transformation. This app is running off a node back-end which allows me to run it on heroku. After getting introduced to Webpack via this project, I have now adapted it to run my entire build process. The config file is quite different from the streams and .pipe() commands used in gulp, but it is a way that has me focused on writing code and not setting up the environment.

Simon Game -> Code On GitHub

My Simon Game project. For this project I used the module pattern with ES6 import to separate the JavaScript files which really helped me when it came to testing with Jasmine. Learning that I can write my test files in ES6 and not have to transpile to ES5. The trouble with this project came when I was thinking of how to 'listen' for the users click and then test that to the played sequence. At first I thought of using a setTimeout() but could not think of where to go from there. It came to me to just run a test() function after the user clicks a panel and compare the two arrays. If they differ then there is a fail. On the other hand if they are the same length and match then advance to next level.

Apartment Service Calculator CEC 2012 -> Code On GitHub

The idea for this application came to me during my time in electrical trade school level 2. In Code class there are packages given out as homework to complete over a couple of weeks. There was a concern raised that if a student is to finish the first package and makes a mistake, it will likely carry onto the next homework assignments since the same electrical code rules will apply. This app will take away that worry by allowing the apprentice to verify they have the right answer before moving on to the next. However, the fun does not stop there, as any electrical company could and should benefit from the efficiency of this application for calculating the total demand for an apartment building.

Pomodoro Timer -> Code On GitHub

My first app using Bootstrap. The ability to have something handle the basic layout and setup of DOM elements really helped me get to writing the JavaScript code faster. The project explores the world of setInerval() and how one goes about dealing with pausing the interval and restarting it all together. During my time on the Free Code Camp Chat help rooms I find the biggest struggle campers have on this one is having the setInerval() id assigned to a variable that is scoped to a function. Leading to the inability of stopping the interval by calling clearInerval(). After reading through Douglas Crockford's JavaScript The Good Parts I realized I should have used a better way of handling the global variables. The plan could have been as simple as have one global object to contain the variables I want to access from anywhere.
