Projects

Apartment Service Calculator CEC 2012

The idea for this application came to me during my time in electrical trade school level 2. In Code class there are packages giving out as homework to complete over a couple of weeks. There was a concern raised that if a student is to finish the first package and makes a mistake, it will likely carry onto the next homework assignments since the same electrical code rules will apply. This app will take away that worry by allowing the apprentice to verify they have the right answer before moving on to the next. However, the fun does not stop there, as any electrical company could and should benefit from the efficiency of this application for calculating the total demand for an apartment building.

Pomodoro Timer

My first app using Bootstrap. The ability to have something handle the basic layout and setup of DOM elements really helped me get to writing the JavaScript code faster. The project explores the world of setInerval() and how one goes about dealing with pausing the interval and restarting it all together. During my time on the Free Code Camp Chat help rooms I find the biggest struggle campers have on this one is having the setInerval() id assigned to a variable that is scoped to a function. Leading to the inability of stopping the interval by calling clearInerval(). After reading through Douglas Crockford's JavaScript The Good Parts I realize I should have used a better way of handling the global variables. The plan could have been as simple as have one global object to contain the variables I want to access from anywhere.
